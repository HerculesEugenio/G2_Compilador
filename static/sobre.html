DS LIVE (DATA STRUCTURE LIVE)

Este projeto foi feito na matéria de compiladores, e consiste em uma aplicação que permite a manipulação de pilhas utilizando comandos de uma linguagem personalizada. Ele combina algumas tecnologias, como Java, com a criação de gramáticas personalizadas por meio do ANTLR.

Os comandos disponíveis incluem:

  • Criar Pilha: Defina novas pilhas com identificadores únicos.
  • Adicionar (Push): Insira itens no topo da pilha.
  • Remover (Pop): Retire itens do topo da pilha.
  • Apagar Pilha (Delete): Exclua pilhas criadas.
  • Mostrar o topo (Get_Top): Mostre o último item da pilha.

O objetivo é demonstrar a integração de um interpretador de linguagem com uma interface gráfica, apresentando as pilhas de forma visual e interativa.
